Before You Hand In the Laptop

Please complete these steps before bringing the laptop to school.

Back up first. The laptop will be reset as part of the setup process. Existing files, apps, user accounts and settings will be removed.

Back up anything important

Save files to a personal cloud account, USB drive, external hard drive or another computer. The school cannot take responsibility for personal files left on the device before setup.

Include the charger

The charger must be handed in with the laptop. Please clearly label the charger with the student's name and class.

Label the laptop or bag

Include the student full name, class or tutor group if known, parent contact name and parent contact email. Please do not put decorative stickers on the outside of the laptop.

Write down the password, if there is one

If the laptop already has a password or PIN, please include it clearly on the label or inside the laptop bag. Missing or incorrect passwords may delay setup.

Check the laptop powers on

Please check the laptop turns on, the charger works, the screen is not damaged, the keyboard and trackpad work, and the stylus is included if already purchased.

Complete the device submission form

The device submission form link will be added nearer the time. Please complete it before handing in the laptop.
